Data Science: Disease Outbreak Analysis

Group Members

  • Joel Daniel Rico
  • Melissa Huynh

Description of Work

This project will use data science concepts to visualize various statistics relating to the spread of a disease. We will research the causes, identify a reliable source of data on disease spread, and practice simple deep learning algorithms. We will then clean and prepare the data for visualization, build an accessible website, and visualize the data in a map or another type of visualization.

Tools and Resources

  • Data science concepts
  • Articles on disease predictions
  • Reliable source of data on disease spread
  • Data cleaning and preparation tools
  • Web development framework (SvelteKit)
  • Data Visualization tools

Enjoy!
